About Me

My name is Jonathan and I am a software engineer based in Ireland. I go to the Institute of Technology Carlow for my undergraduate studies in Software Development, and will be expected to graduate in 2022. I enjoy designing neat order and control structure of code and I have very high attention when it comes to even the smallest details. I have done two large projects on campus and several small personal projects. Aside from programming I also enjoy drawing, painting, and illustration artwork in my spare time.

Software Projects

ProjectOneSky: Sentiment Analysis

This project serves as my academic undergraduate final year project and is a machine learning implementation of sentiment analysis for review texts, commissioned by Project One Sky, a wellbeing education company providing wellbeing courses for secondary school students across Ireland. Main technology stack include Python, Keras TensorFlow, Flutter, Dart, Firebase, and hosted on Google Cloud Platform.

Optician Shop Stock Control System

End of year project for my second year at IT Carlow. Created a stock control system for a mock online store selling optical products. Technology stack include PHP, HTML5, CSS3, JavaScript, SQL and the MySQL database system
